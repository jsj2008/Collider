To install and use AppEasy for Marmalade please follow these instructions:
1) Please open the zip AppEasyMarmalade.zip
2) Unzip this zip file to c:
3) Run c:\AppEasy\AppEasy.exe this will launch the AppEasy for Marmalade project manager
4) Open a project within AppEasy by clicking the Open button then navigate to the c:\AppEasy\examples folder and select an example
5) Press the Create MKB button to generate an MKB file and open the project in Visual Studio
6) Hit eth debug button to compile and run the example

Eventually all examples will be added to the SDK as separate projects

Also note that the Apple Wizard is located in c:\AppEasy\Tools\AppleWizard.exe
